## 🖥️ What codbash does

codbash is a browser dashboard for Claude Code and Codex sessions. It helps you keep track of your AI coding work in one place.

With codbash, you can:

- view your sessions in a clean browser view
- search past chats and prompts
- resume work from older sessions
- tag sessions for quick sorting
- manage many sessions without losing track

It is built for people who want a simple way to review AI coding sessions on Windows.

## 🛠️ Troubleshooting

### The app does not open
- Check that the download finished.
- Try running the file again.
- Restart Windows and try once more.

### I cannot see my sessions
- Refresh the dashboard.
- Check that the session source is connected.
- Open an older session file or source again.

### Search returns no results
- Try a shorter search term.
- Check spelling.
- Search by tag or date instead of full text.

### Windows blocks the file
- Right-click the file and choose Run as administrator.
- Check your security settings.
- If the file is in a zip folder, extract it first.

### The browser page looks blank
- Refresh the page.
- Try another browser.
- Clear the browser cache for the site.
